Census Analysis Apps


Parse Data

This module restructures the format of Excel data tables from the Census website to be more easily analyzed (either by the analyst or using the Analyze Census Data module). This module reformats the tables to have the Census tract or block group as rows (instead of as individual columns) and the characteristic variables and associated estimates as corresponding rows.

Descriptive Analysis

This module inputs a zipped folder of parsed Census Excel files (outputted from the Parse Census Data module) and outputs a single Excel file with data tabulated from each parsed file. The module can also compute percentages and percentage ranks of selected characteristics, correlation matrices showing relationships between characteristics, box plots showing distributions, and other descriptive statistical summaries.

Advanced Analysis

The advanced analysis module pulls data from the Census API and allows users to plot two variables against each other to analyze relationships. Associated R-squared values and linear regression equations are shown.
