Census Data App

This application parses and analyzes data from the US Census Bureau website.

Module 1. Parse Data

  • Step 1:
    • Download table files from the Census website with one or more of the table IDs shown here. Download a table with the geography set as by tract or blockgroup and download the file in Excel format.

    Overview

    This module restructures the format of Excel data tables from the Census website to be more easily analyzed (either by the analyst or using the Analyze Census Data module). This module reformats the tables to have the Census tract or block group as rows (instead of as individual columns) and the characteristic variables and associated estimates as corresponding rows (e.g., male/female, income range, etc). This makes the file much easier to work with, whether by using a pivot table in Excel, connecting to GIS shapefiles, or importing into a Power BI dashboard.
