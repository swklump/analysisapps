Traffic Analysis Apps


AKDOT Traffic Data

This application reorganizes traffic volume and speed data from the Alaska Department of Transportation and Public Facilities website to be more easily analyzed and combines multiple data files into one continuous table in Excel.


IHSDM Optimization

This module transfers data from HTML files in IHSDM to an Excel format that can be easily analyzed. The module compiles evaluation output data from all elements (freeway segments, ramps, etc.) together so that the entire network can be analyzed and compared with ease.


Trip Assignment Model

The trip assignment model assigns trips in origin-destination pairs to the least travel time paths in a network. Total volumes by link are computed and are shown color-coded in a map.
