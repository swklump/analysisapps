IHSDM Optimization App

This application parses and analyzes HTML output files from the IHSDM crash prediction modeling software (www.ihsdm.org).

Module 1. Parse Data

  • Step 1:
    • Upload an evaluation output HTML file or zipped folder of HTML files from IHSDM.
    {% csrf_token %}

    After clicking the "Submit" button, a zipped folder named "ihsdm_parsed_files.zip" will appear in the "Downloads" folder on your computer. This module may take up to a minute to complete if parsing hundreds of files.

    Overview

    This module transfers data from HTML files in IHSDM to an Excel format that can be easily analyzed. The module compiles evaluation output data from all elements (freeway segments, ramps, etc.) together so that the entire network can be analyzed and compared with ease. The module creates a separate Excel file for each element type and a separate tab in each Excel file for each evaluation table type.
